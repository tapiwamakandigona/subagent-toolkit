# Research Report Template

Copy this skeleton and fill it in. Delete sections that genuinely don't apply — don't pad.

```markdown
# <Question being answered, phrased as a question>

**Date:** <research date> · **Scope:** <what was and wasn't investigated>

## TL;DR
- <Direct answer to the core question, one sentence>
- <Most decision-relevant finding> ✅
- <Second finding> ✅
- <Important caveat or disputed point> ⚠️/❌

## Findings

### <Sub-question 1>
<2–6 sentences. Lead with the answer, then the evidence. Inline-cite every
number and claim: "... grew 34% YoY [1]."  Mark confidence: ✅ confirmed,
⚠️ single-source, ❌ disputed.>

### <Sub-question 2>
...

## Where sources disagree
| Claim | Position A (source) | Position B (source) | Likely explanation |
|---|---|---|---|

## Open questions
- <What couldn't be verified, and what it would take to verify it>

## Sources
1. <Title> — <Publisher>, <date>. <URL> (primary/secondary)
2. ...

## Method (optional, for auditability)
<Search passes performed, notable dead ends, anything that biased coverage.>
```

## Quality checklist before submitting

- [ ] TL;DR answers the *original* question, not a nearby easier one
- [ ] Every number has a citation and a date
- [ ] At least one adversarial search was run per major claim
- [ ] Single-source claims are visibly flagged
- [ ] Disagreements are reported as disagreements, not silently averaged
- [ ] Open questions section is honest (empty only if truly nothing is open)
